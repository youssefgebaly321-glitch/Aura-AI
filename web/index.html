🥷 Stealth Professional Tip

Always interact with the app using Global Hotkeys. Clicking on the window during a proctored exam may trigger focus-detection alerts from monitoring software.

📸 Multi-Screenshot Analysis

You can capture multiple screenshots with Alt+S to build a queue, then process them all at once with Alt+P. This is highly recommended for complex multi-part coding questions.

⚡ Speed Strategy

If your primary AI model is lagging, use Alt+E to immediately trigger Auto-Select. Aura will switch you to the fastest healthy provider available.
